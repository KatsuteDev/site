BackgroundX

Background image extension for VSCode web and GitHub Codespaces

Report an Issue • Chrome Extension • Mozilla Add-on

Installation

Chrome (recommended)

Firefox

requires Firefox 121 or higher

Features

Multiple Backgrounds

Add background images for the whole window, editors, sidebars, or the panel.

Full Window

Editor, Sidebar, and Terminal

Animated Backgrounds

GIF image support.

Transition

Transition between multiple background images.
